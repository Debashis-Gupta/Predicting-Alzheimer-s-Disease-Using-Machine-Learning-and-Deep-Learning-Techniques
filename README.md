
## Requirements
- Python 3.x
- Required libraries (e.g., pandas, numpy, scikit-learn, torch.)

## Installation
1. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the following command
```
python3 main.py
```
